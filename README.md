# Daily Reflection Tree

This project implements a deterministic reflection system using a structured decision tree.

## Overview
The system guides users through an end-of-day reflection based on three axes:
- Locus (Victim vs Victor)
- Contribution (Contribution vs Entitlement)
- Perspective (Self vs Others)

## Key Features
- Fully deterministic (no AI at runtime)
- Fixed decision paths
- Structured branching logic
- Psychological grounding

- reflection-tree.json → Core tree logic
